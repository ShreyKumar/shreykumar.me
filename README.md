# shreykumar.me

A static portfolio site built with Next.js App Router

## Quickstart
First, clone the repository
```
git clone https://github.com/ShreyKumar/shreykumar.me
```
Then install all dependencies
```
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```
Then, run the dev server
```
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
This will start the dev server on port 3000

## Hosting
The site is hosted on Vercel with a CI/CD pipeline fully integrated. Each commit will trigger a new build and comitting to `main` will automatically push to production and deploy the site.
